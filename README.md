# Stock-Price-Prediction-using-Linear-Regression
This project is a beginner-friendly implementation of stock price prediction using Machine Learning. It uses historical stock data fetched from the Yahoo Finance API and applies a Linear Regression model to predict the next day’s closing price based on the current day’s price. This project shows how basic Ai-ml concepts can be applied to real world financial data analysis.

# Features
* Fetch real-time stock data using Yahoo Finance
* Predict next day's closing price
* Visualize actual vs predicted prices
* Simple and easy-to-understand code

## Machine Learning Approach
* model used : Linear Regression
* library : Scikit-learn

Features used:
* Current day closing price
  
Target variable:
* Next day closing price
 
Working principle:
* The model learns the relationship between today's stock price and following day's price using historical data patterns.

Output
* Predicted next day price
* Visualization graph comparing actual and predicted price.
